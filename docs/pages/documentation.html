Introduction

Extendx is an open-source Go based application designed to help streamers do screen sharing between devices/platforms without the need for a capture card.

Server

The server is a screen capture that you want to share on the client side based on IP address.

Client

Client is the one who takes the screenshot from the server based on the ip
